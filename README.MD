## Запуск проекта

1. Клонируем репозиторий `git clone link-to-repo`
2. Устанавливаем зависимости: `npm install`
3. Настроить переменные окружения в .env файле: пример в .env-example
4. Загружаем миграции в базу данных: `npm run migrate`
5. Загружаем тестовые данные в БД: `npm run seed`
6. Запуск проекта `npm run dev` - запускает backend
